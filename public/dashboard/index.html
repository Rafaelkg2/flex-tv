Sem transmissão ativa
Como usar:
1. Baixe o Mediamtx e execute mediamtx.exe no laptop.
2. No OBS, configure: Custom RTMP → rtmp://localhost:1935/live/main.
3. Cole a URL HLS acima (substitua o IP pelo do seu laptop).
4. Clique em Iniciar transmissão ao vivo.

Consulte mediamtx/README.md para o guia completo.
Nota técnica: As TVs recebem a URL HLS e tocam o stream com a tag <video> nativa. Nenhum software é necessário nas TVs. A URL HLS deve ser acessível pela rede interna da Grupo Flex (mesmo Wi-Fi que as TVs e o laptop do operador).
